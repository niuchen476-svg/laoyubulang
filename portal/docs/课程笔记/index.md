## 目录结构规划

```
课程笔记/
├── index.md          ← 本页
├── 数学/
├── 编程基础/
└── 专业课/
```

## 如何添加笔记

1. 在 `课程笔记/` 下新建子目录（如 `Python/`）
2. 创建 `index.md` 作为该模块入口
3. 在 `mkdocs.yml` 的 `nav` 中添加对应条目
4. 提交推送，CI 自动部署

!!! tip "PDF 笔记内嵌"
    如需在页面内展示 PDF 扫描件，参考[首页说明](../index.md#pdf)的嵌入方法。
